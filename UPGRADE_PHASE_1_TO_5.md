## 📋 قدم‌های اجرا

### قدم ۱: کپی همه فایل‌ها
محتوای ZIP جدید را روی پروژه فعلی **Replace** کنید.

### قدم ۲: Restore پکیج‌ها
- راست کلیک روی Solution → **Restore NuGet Packages**

### قدم ۳: حذف Migration قبلی و دیتابیس
موجودیت‌های زیادی اضافه شده. بهترین کار:

در SSMS:
```sql
DROP DATABASE SchoolManagement;
```

پوشه `Migrations` در `SMS.Infrastructure` را پاک کنید.

### قدم ۴: Migration جدید
در Package Manager Console:
- Default Project = `SMS.Infrastructure`

```powershell
Add-Migration InitialCreate -StartupProject SMS.Web
Update-Database -StartupProject SMS.Web
```

### قدم ۵: اجرا
F5 → ورود با `admin` / `Admin@123`

## 🎯 جریان کار سیستم

برای استفاده کامل، این ترتیب رو دنبال کنید:

### 1. تنظیمات اولیه (یکبار)
1. **افزودن مدرسه** → Schools / Create
2. **افزودن کلاس** → Classrooms / Create
3. **افزودن دانش‌آموزان** و **ثبت‌نام در کلاس**
4. **افزودن کارکنان** → Staff / Create
5. **تخصیص کارکنان به مدارس** → Staff / Details → سمت معلم یا معاون

### 2. تعریف دروس
1. **دروس کلی** → Subjects (ریاضی، علوم، ...)
2. **دروس پایه** → Subjects / Grade → برای هر پایه ضریب و وزن تنظیم کنید

### 3. تخصیص معلم به درس کلاس
1. در صفحه کلاس‌ها → دکمه **📚 دروس**
2. درس + معلم رو انتخاب کنید

### 4. عملیات روزانه
- **حضور و غیاب** → Attendance → انتخاب کلاس → ثبت
- **آزمون** → از صفحه دروس کلاس → دکمه آزمون‌ها → ایجاد + نمره‌دهی
- **انضباطی** → Discipline → ثبت رکورد

## 🆕 ویژگی‌های کلیدی این نسخه

### 📝 ارزشیابی توصیفی + عددی
- موقع ساخت آزمون انتخاب می‌کنید: عددی یا توصیفی
- برای توصیفی، مقیاس انتخاب می‌شود (مثلاً ۴ سطحی)
- صفحه ورود نمرات خودکار با نوع آزمون تطبیق می‌کند

### 🔒 نهایی‌سازی آزمون
- بعد از ثبت همه نمرات، آزمون رو **نهایی** کنید
- بعد از نهایی شدن، دیگه قابل تغییر نیست (امنیت داده)

### 👨‍🏫 یک معلم در چند مدرسه
- در پروفایل هر کارمند، می‌تونید چند تخصیص اضافه کنید
- مثلاً: معلم در مدرسه الف + معاون در مدرسه ب

### 📊 گزارش حضور و غیاب
- آمار حاضر/غایب موجه/غیرموجه/تاخیر در بازه زمانی
- قابل فیلتر بر اساس تاریخ شمسی

### ⭐ تشویق و تنبیه
- ۹ نوع پیش‌فرض (تشویق کلاسی تا تعلیق از تحصیل)
- تاثیر خودکار در نمره انضباط
- امکان اطلاع‌رسانی به اولیا

### 🌱 Seed Data کامل
- ۲۶ درس پیش‌فرض
- مقیاس توصیفی ۴ سطحی
- ۹ نوع آزمون
- ۹ نوع رخداد انضباطی
- نوبت‌های اول و دوم
- ۵ استان + ۱۰ شهر

## 🐛 رفع مشکلات احتمالی

### اگر Migration خطا داد:
```powershell
Remove-Migration -StartupProject SMS.Web
```
بعد دوباره Add کنید.

### اگر در صفحه نمرات، اعداد ذخیره نشد:
- مطمئن شوید آزمون **نهایی نشده** (در حالت نهایی، فیلدها قفل می‌شوند)

### اگر در صفحه دروس کلاس، معلمی نشد انتخاب کرد:
- ابتدا باید کارکنان رو با سمت "معلم" به مدرسه تخصیص بدید

### اگر حضور و غیاب ثبت نشد:
- مطمئن شوید دانش‌آموزان در کلاس **ثبت‌نام فعال** دارند
